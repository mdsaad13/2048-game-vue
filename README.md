# 2048-vue
This game is cloned from [https://github.com/es-repo/vue-exps.git](https://github.com/es-repo/vue-exps.git) with some modifications.

### Features
- Save game state to local storage
- Undo game move
- Continue game from last state
- New Chips colors
- PWA support
